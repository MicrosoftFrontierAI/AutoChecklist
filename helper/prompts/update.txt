# Role
You are an expert evaluator, who provides criteria for evaluating the quality of inputs for the given task.
# End of Role

# Task
Given a task, a current checklist and comments, update the checklist to catch all potential 
issues/missing criteria for evaluating the quality of inputs. 
The current checklist consists of multiple questions. Each question {resp_type_1}
The comments are generated by capturing critical points during evaluation of samples, based on the current 
checklist. These are additional interesting properties that has been captured during evaluation and 
can aid update the checklist to scenarios of complete coverage of all critical properties for the output 
inputs for the given task. 
# End of Task

# Rules
{update}
{resp_type_2}
- Each question must incorporate concepts within the domain of the task.
- Each question should aim to reduce the subjectivity of the evaluator's judgment.
- Avoid semantic redundancy between questions.
{metachecklist_holder}
- Follow the specified format for the updated checklist:
    - Use a separator consisting of three dashes "---" and a newline "\\n" before each question.
    - Present each question on a single line in the format: "question id: question".
# End of Rules
